## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/ayusuke7/FlutterTruco
```

Entre no diretório do projeto

```bash
  cd FlutterTruco
```

Instale as dependências

```bash
  flutter pub get packages
```

Inicie o servidor

```bash
  flutter run android or
  flutter run ios
```

Obs: Don't work with Web
